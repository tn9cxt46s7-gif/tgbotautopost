```

✅ Посты от твоего имени
✅ Антибан: интервалы, лимиты, вариации текста
✅ Несколько групп и объявлений
✅ Оплата: Stars / CryptoBot / карта
✅ Пробный день бесплатно
✅ Саппорт @eb_support

Промокод START20 — скидка 20% на первую оплату

👉 Напиши боту: @ТВОЙ_БОТ
```

## Советы по маркетингу

1. **Оффер в первой строке** — боль («руками кидать») + результат («само»).
2. **Соц.доказательство** — скрины «ушло в N групп», отзывы в закрепе.
3. **Пробный день** — снижает страх оплаты; веди в «Профиль → Пробный день».
4. **Дефицит** — «VIP30 только на 30 человек».
5. **Куда лить трафик:** чаты продавцов, барахолки (осторожно с правилами), Avito-селлеры, TikTok/Reels «день из жизни селлера».
6. **Не обещай «0 банов»** — пиши «снижаем риск», иначе будут возвраты.
7. **CTA один** — ссылка на бота + промокод.
8. **Креатив:** лицо/товар + телефон с Telegram + текст «автопост».

## Что писать в закрепе бота

```
Автопостинг объявлений в барахолки от твоего аккаунта.
/start → пробный день или Подписка.
Промо: START20
Саппорт: @eb_support
```

---

## Воронка

1. Рекламный пост → бот  
2. /start → пробный день  
3. Подключить аккаунт + 1 группа + 1 объявление  
4. «Запостить сейчас» → вау-эффект  
5. Подписка (Stars/CryptoBot) + промокод  
6. Рефералка (+3 дня)
