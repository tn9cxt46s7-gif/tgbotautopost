# Roadmap — что можно добавить в следующих обновлениях

Уже в v2.1: посты от аккаунта клиента, антибан, Stars + карта/крипта (pending),
саппорт @eb_support, пробный день, инструкция, статистика в профиле.

## Быстрые продажи (высокий ROI)

1. **Автовыдача после ЮKassa / CryptoBot** — вебхук оплаты без ручного «подтвердить»
2. **Промокоды** — скидки и партнёрские коды
3. **Мультиязычность** RU/EN — шире рынок
4. **White-label** — свой бренд/токен для реселлеров
5. **Готовые шаблоны объявлений** по нишам (телефоны, авто, одежда)

## Продукт и удержание

6. Напоминание за 2 дня до конца подписки
7. История постов с фильтрами (успех / ошибка / группа)
8. Чёрный список групп (глобально админом)
9. Дашборд веб для админа (выручка, churn)
10. Экспорт объявлений / бэкап сессии

## Техника и надёжность

11. Redis FSM вместо MemoryStorage (критично для Vercel scale)
12. Очередь постов (Celery / RQ) вместо cron-тика
13. Health-мониторинг + алерт админу если posting_cycle падает
14. Тесты pytest на оплату, антибан, лимиты
15. Docker Compose: bot + postgres + worker

## Монетизация

16. Тариф «агентство» (много аккаунтов одного клиента)
17. Pay-as-you-go: оплата за N постов
18. Партнёрка % с платежей (не только +дни)
